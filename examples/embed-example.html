Rainbow Relax Widget Embed Example

This page demonstrates how to embed the Rainbow Relax widget using the recommended script pattern.

Implementation Code

Copy and paste this code into your website to embed the Rainbow Relax widget: 

<!-- Rainbow Relax Widget Container -->
<div id="rainbow-relax-container" style="width: 500px; height: 500px;"></div>

<script>
  (function(w, d, s, src, id) {
    if (d.getElementById(id)) return;
    var js = d.createElement(s);
    js.id = id;
    js.src = src;
    js.async = true;
    d.head.appendChild(js);
  })(window, document, "script", "https://cdn.example.com/widget/rainbowRelax.js", "rainbow-relax");

  window.myWidgetConfig = {
    showQuickExit: false,
    donateURL: 'https://www.paypal.com/donate/?hosted_button_id=G5E9W3NZ8D7WW',
    getHelpURL: 'https://www.thetrevorproject.mx/ayuda/',
    width: '500px',
    height: '500px',
    containerId: "rainbow-relax-container",
    cdnBase: "https://cdn.example.com/widget/sounds/",
    audioEnabled: true
  };
</script>
